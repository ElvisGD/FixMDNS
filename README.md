HOW TO via Mobile:

1: Envoi du lien https://github.com/KDOGD/FixMDNS/blob/main/FixDnscache_Universal.7z par SMS.

2: L'utilisateur appuie sur les 3 petits points en bas et appuie sur Download et confirme en appuyant sur Télécharger.

3: L'utilisateur connecte un câble USB du téléphone vers son appareil. Ouvre les notifications et appuie sur USB. Puis sélectionne Transfert de fichier.

4: Le PC détecte un nouveau lecteur dans "Ce PC". Le fichier est disponible dans Stockage -> Download.

5: Clique droit -> Couper -> Coller dans C:\Temp

6: Clique droit -> Extraire Tout.

7: Clique droit sur le fichier .bat -> Executer en tant qu'administrateur...
